hCaptcha Verifier

Base domain: https://hcaptcha-verifier.divideprojects.com/

There are just 2 URL Parameters that are required:
- token: the response token given after solving the captcha.
- secret_key: the secret key given by the hCaptcha on settings page of account.

A simple example looks like:
https://hcaptcha-verifier.divideprojects.com/siteverify?token=yourtokenhere?secret_key=yoursecretkeyhere

This will check the captcha with response 'yourtokenhere' if it's valid or not.

Source for this service is available here:
